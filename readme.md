# OOP Concepts 

This repository contains basic implementations of Object-Oriented Programming (OOP) concepts using code examples.

# Topics Covered

- Classes & Objects  
- Constructors & Destructors  
- Encapsulation  
- Inheritance  
- Polymorphism  

# About

Object-Oriented Programming (OOP) is a programming paradigm based on the concept of objects.  
It helps in organizing code, making it reusable, and easier to maintain.
